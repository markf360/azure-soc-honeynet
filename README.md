# Implementing a SOC and Honeynet in Azure

## Overview
This project simulates a basic Security Operations Center (SOC) and honeynet environment using Microsoft Azure.

## Tools & Technologies
- Microsoft Azure
- Ubuntu Virtual Machines
- Microsoft Sentinel (SIEM)
- Log Analytics
- Tenable.io

## What I Did
- Deployed Azure virtual machines, including Ubuntu systems.
- Simulated a SOC and honeynet environment.
- Identified CVEs using Tenable.io and assessed severity using CVSS scoring.
- Collected and analyzed security logs using Microsoft Sentinel and Log Analytics.
- Investigated system activity on Ubuntu systems using Linux command-line tools.

## Key Takeaways
- Learned how SOC workflows operate in a cloud environment.
- Gained hands-on experience with SIEM monitoring and vulnerability analysis.
